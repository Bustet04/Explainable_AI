# Raw Data Directory

This directory is for storing raw physiological signals and datasets.

## Sleep Analysis Data
- Place Sleep-EDF database files here (excluded from repo due to size)
- Download from: https://physionet.org/content/sleep-edfx/1.0.0/

## Mental Disorders Data
- Mental disorder EEG data is stored in `data/mental_disorders/`
- Includes EEG.machinelearing_data_BRMH.csv

**Note**: Large raw data files are excluded from version control via `.gitignore`
