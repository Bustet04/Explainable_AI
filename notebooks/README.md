# Notebooks Directory

This directory contains clean, documented Jupyter notebooks for model training and analysis.

## Subdirectories

### sleep_analysis/
Sleep stage classification notebooks:
- `train_sleep_classifier.ipynb`: Supervised MLP classifier (87% accuracy)
- `train_unsupervised_sleep_clustering.ipynb`: Unsupervised clustering with explainability

### mental_disorders/
Mental disorder classification notebooks:
- `train_binary_randomforest.ipynb`: Random Forest for addictive disorder detection
- `train_engineered_features.ipynb`: Enhanced feature engineering approach
- `train_neural_network.ipynb`: Deep learning model for mental disorder classification

## Usage
Each notebook is self-contained with:
- Clear documentation and explanations
- Data loading and preprocessing steps
- Model training and evaluation
- Explainability analysis (XAI)
- Results visualization
